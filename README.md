# zhweb-jiuyou-guide-d294

本仓库用于归档和发布多个独立 HTML 页面。页面内容涉及网页技术相关的资料整理、使用说明或参考指南，不针对任何单个域名或具体网站。

## 项目简介

这是一个以 HTML 页面形式存放的文档/工具集合。仓库中的每个页面均为独立文件，可直接在浏览器中打开查看。项目不提供动态服务或后端支持，所有页面均为静态内容。

## 目录说明

```
/
├── README.md          # 本文件
├── index.html         # 项目入口页面（可选）
├── pages/             # 存放主要 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 静态资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：放置独立 HTML 页面，每个文件对应一个功能或主题。
- `assets/`：存放页面引用的外部资源，如样式表、JavaScript 脚本和图片。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文档，可脱离仓库运行。
- 页面之间可能通过超链接相互引用，但无强制依赖关系。
- 归档内容主要围绕网页技术相关主题，包括但不限于配置参考、使用步骤、常见问题等。
- 所有页面均采用中文编写，部分页面可能包含代码示例或截图。
